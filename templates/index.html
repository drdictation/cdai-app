CDAI PDF Generator

How to Use This Tool

This tool automates the process of filling out CDAI PDF forms for Medicare/PBS applications. Follow the steps below carefully to ensure your files are processed correctly.

Step 1: Prepare Your Data File

Your patient data must be in an Excel (.xlsx) or CSV (.csv) file. The first row must be a header row with the columns: firstname, lastname, Birthdate, Medicare number, Height, Weight, `Infusion location (must include hospital provider number)`, and `gastroenterologist`.

Step 2: Name Your Blank PDF Templates

For each gastroenterologist, you need a corresponding blank CDAI form in PDF format. The file MUST be named using the doctor's last name.

The format is `CDAI_BLANK_Lastname.pdf`.

Step 3: Generate & Check Results

Use the tool in the "Generator" tab. A `.zip` file will be downloaded containing the filled PDFs and a `processing_report.txt` file that details which rows succeeded and which were skipped (and why).
